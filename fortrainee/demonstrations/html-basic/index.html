Mozilla Firefox

About Mozilla Firefox

Mozilla Firefox (or simply Firefox) is a free and open-source web browser developed by the Mozilla Foundation and its subsidiary the Mozilla Corporation.

Availability

Firefox is available for several Operating systems:

  • Windows
  • macOS
  • Linux
  • Android (formerly Firefox for mobile)
  • Firefox OS (discontinued)

CSS Prefix

Mozilla supports a number of extensions to CSS that are prefixed with -moz-. For example: 

        -moz-column-count
        -moz-column-fill
        -moz-column-gap

This is a quotation taken from the Mozilla Developer Center.
